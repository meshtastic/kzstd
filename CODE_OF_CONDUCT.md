# Code of Conduct

This project follows the
[Meshtastic Code of Conduct](https://meshtastic.org/docs/community/code-of-conduct/),
which adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
v2.1.

In short: be respectful, welcoming, and constructive. Harassment, discrimination,
and abusive behavior are not tolerated, in this repository or in any community
space associated with the project.

## Reporting

Report unacceptable behavior to the Meshtastic community moderators through the
channels listed at [meshtastic.org](https://meshtastic.org). Reports are handled
confidentially.
